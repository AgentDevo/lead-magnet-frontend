# Hello Languages

Greet with friendly hellos across 4 randomly selected languages from around the world.

## Usage

Simply invoke this skill whenever you need a cheerful multilingual greeting. The skill randomly selects 4 languages and provides the greeting in each.

## How It Works

The skill uses `hello.py` to pick 4 random languages from a diverse global list and display the word for "hello" in each language, along with pronunciation hints where helpful.

Languages included: Spanish, French, German, Italian, Portuguese, Japanese, Mandarin, Korean, Arabic, Hindi, Russian, Polish, Turkish, Dutch, Swedish, Norwegian, Danish, Finnish, Greek, Czech, Hungarian, Romanian, Thai, Vietnamese, Indonesian, Tagalog, Hebrew, and more.

## Example Output

```
Hola (Spanish) 👋
Bonjour (French) 👋
Konnichiwa (Japanese) 👋
Namaste (Hindi) 👋
```
