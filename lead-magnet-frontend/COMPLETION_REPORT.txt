INTEGRATION POINTS
================================================================================

All API endpoints pre-configured and ready to connect:

Auth:
  - POST /auth/signup
  - POST /auth/login
  - POST /auth/logout

Lead Magnets:
  - POST /lead-magnets (create)
  - GET /lead-magnets (list)
  - GET /lead-magnets/:id (read)
  - PUT /lead-magnets/:id (update)
  - DELETE /lead-magnets/:id (delete)

AI:
  - POST /ai/generate-content
  - POST /ai/generate-pdf

Analytics:
  - GET /analytics
  - GET /analytics/leads/:magnetId

User:
  - GET /user/profile
  - PUT /user/profile
